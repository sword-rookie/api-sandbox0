# Current Technology Stack

## Frontend (User Interface)
- **Framework:** Next.js (Pages Router)
- **Library:** React.js
- **Styling:** Tailwind CSS (with custom theming for dark cybernetic mode)
- **Node Graph Engine:** `@xyflow/react` (React Flow) for the high-performance Blueprint Editor
- **Icons:** Material Symbols (Google Fonts) & Lucide React

## Backend (Orchestrator)
- **Language:** Golang (Go)
- **Routing:** `gorilla/mux`
- **Build Engine:** Nixpacks (Zero-config OCI image builder)
- **Containerization Management:** Direct communication with the Docker Daemon (`/var/run/docker.sock`)

## AI / Infrastructure
- **Local LLM Engine:** Ollama (running locally as a containerized service)
- **Orchestration:** Docker Compose
